### Run Locally

To run **Dami** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Rotimi1234/dami.git
```

Windows:

```bash
git clone https://github.com/Rotimi1234/dami.git
```
